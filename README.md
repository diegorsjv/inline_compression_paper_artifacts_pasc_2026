# Inline Lossy Compressed MPI Communications in GPU-based Plasma Turbulence Simulation

## Overview 

This repository holds data and code artifacts for the different sections of the paper submission "Inline Lossy Compressed MPI Communications in GPU-based Plasma Turbulence Simulation"
for the PASC Conference 2026. 

Each subdirectory in this repository includes a README.md file describing the directory structure and contents. 

For questions regarding the different artifacts and their processing, please contact the corresponding author.
